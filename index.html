Introducción

Android es un sistema operativo para dispositivos móviles de código abierto, desarrollado por Google que fue lanzado el año 2008. Contiene distintos lenguajes, entre estos está JAVASCRIPT, C y C++, es de carácter colaborativo, es decir, cualquier usuario de Android puede crear aplicaciones y también cualquier usuario puede acceder a los códigos de estas aplicaciones.
